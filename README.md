# MyTool

A professional Python-based network scanning and system information tool.

## Features

- Host Discovery
- Port Scanning
- OS Detection
- Full Network Scan
- System Information
- Interactive Terminal Menu

## Requirements

- Python 3.10+
- Linux (Kali Linux Recommended)

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/MyTool.git
cd MyTool
chmod +x install.sh
./install.sh
```

## Usage

Run with Python:

```bash
python3 ku_netscan.py
```

Or after installation:

```bash
mytool
```
